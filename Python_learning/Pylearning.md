## python是面向对象的解释性高级语言
## python是强类型的动态脚本语言
```
# ctrl d快捷复制本行到下一行；ctrl /快捷注释和去掉注释；三对双/单引号是多行注释；#是单行注释
print("a","b",sep="\n")  # 会输出一上一下两个字母，如果不指定分隔符就是默认空格
# end默认是换行符\n也就是说打印完上一行内容之后换行打下一个print内容，修改之后就会变成Hello World,Hello World
print("Hello World",sep="\n",end=",")
print("Hello World")
```
